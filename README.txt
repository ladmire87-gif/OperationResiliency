ValorWell Health Clinic Website

- services.html: services overview page
- hormone-replacement-therapy.html: HRT service page

How to edit:
1. Open index.html in a browser to preview.
2. Replace placeholder contact information in each HTML file.
3. Replace SVG files in assets/ with real clinic/provider photos using the same filenames, or update the image paths.
4. Before publishing, have all medical claims, privacy language, and service descriptions reviewed for compliance.
